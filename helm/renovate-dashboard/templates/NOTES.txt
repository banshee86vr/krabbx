---------
The application has been deployed with the following services:

Frontend Service:
  $ kubectl get svc {{ include "renovate-dashboard.fullname" . }}-frontend -n {{ .Release.Namespace }}

Backend Service:
  $ kubectl get svc {{ include "renovate-dashboard.fullname" . }}-backend -n {{ .Release.Namespace }}

To access the frontend from your local machine:
  $ kubectl port-forward svc/{{ include "renovate-dashboard.fullname" . }}-frontend {{ .Values.frontend.service.port }}:{{ .Values.frontend.service.port }} -n {{ .Release.Namespace }}
  
Then open your browser to: http://localhost:{{ .Values.frontend.service.port }}

To access the backend API from your local machine:
  $ kubectl port-forward svc/{{ include "renovate-dashboard.fullname" . }}-backend {{ .Values.backend.service.port }}:{{ .Values.backend.service.port }} -n {{ .Release.Namespace }}
  
Then the API will be available at: http://localhost:{{ .Values.backend.service.port }}
